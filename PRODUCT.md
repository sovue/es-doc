# Product

## Register

product

## Users

Russian-speaking modders of the visual novel "Everlasting Summer" (Бесконечное Лето). Typically arrive mid-project: looking up a specific term, grabbing a code snippet, or understanding how a game mechanic works before writing their mod. Not beginners to the internet, but not necessarily programmers.

## Product Purpose

A documentation hub for mod creation: terminology reference, code examples in context, game resource listings, and call syntax. The site makes scattered community knowledge findable and trustworthy in one place.

## Brand Personality

Precise, warm, community-built. The tone is knowledgeable but not academic — like a senior modder leaving good notes for the next person.

## Anti-references

- Generic ReadTheDocs / Sphinx autogenerated look (sterile, impersonal)
- Confluence-style corporate wiki (too formal, wrong audience)
- Neon/cyberpunk gaming aesthetics (try-hard, distracts from content)

## Design Principles

- Content is the product. Chrome exists only to make text easier to read and navigate.
- Familiar before clever. Modders know wikis; don't surprise them with unusual layouts.
- Warm, not cold. The orange accent is intentional — it should feel like someone's personal craft, not a corporate tool.
- Show, don't narrate. Code examples over explanations wherever possible.

## Accessibility & Inclusion

WCAG AA minimum. Keyboard-navigable sidebar and anchors. No motion-dependent meaning.
